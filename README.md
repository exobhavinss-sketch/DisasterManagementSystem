# Disaster Management System

## Project Description

The Disaster Management System is a Data Structures Using C project designed to manage disaster reports, emergency requests, rescue routes, and resource distribution efficiently.

This project demonstrates the implementation of multiple data structures in a real-world disaster management scenario.

---

# Features

- Add Disaster Reports
- Display Disaster Reports
- Search Disaster Reports
- Delete Disaster Reports
- Update Disaster Reports
- Emergency Request Management
- Rescue Team Hierarchy
- Rescue Route Optimization
- Resource Distribution Queue
- Login Authentication System
- File Handling
- HTML/CSS Frontend

---

# Data Structures Used

| Data Structure | Purpose |
|---|---|
| Linked List | Disaster Reports |
| Stack | Emergency Requests |
| Queue | Resource Distribution |
| Tree | Rescue Team Hierarchy |
| Graph | Rescue Route Optimization |
